# Disguised Stock Monitor
네트워크 설정 도구로 위장한 한국 주식 실시간 모니터링 프로그램

## 개요

`netconfig.py`는 네트워크 인터페이스 상태를 확인하는 것처럼 보이지만, 실제로는 관심 종목들의 주가 정보를 실시간으로 모니터링하는 스텔스 프로그램입니다.

## 주요 기능

- 🔒 **완벽한 위장**: 네트워크 설정 유틸리티로 완전히 위장
- 📊 **실시간 모니터링**: 선택한 종목들의 실시간 주가 정보
- 💰 **직관적 표시**: 네트워크 처리량(Mbps)으로 주가 표시
- 📈 **퍼센트 정보**: 변화율, 고점/저점 대비 퍼센트 표시
- 🎯 **포지션 표시**: 일봉 고저점 범위 내 현재 위치 표시
- 🔧 **관심종목 관리**: 종목 추가/삭제 기능
- 🔍 **종목 검색**: 종목명으로 검색하여 간편 추가
- ⚡ **빠른 업데이트**: 30초 간격 자동 새로고침
- 💾 **설정 저장**: JSON 파일로 관심종목 자동 저장/복원
- 🔄 **백업/복원**: 설정 백업 및 복원 기능

## 데이터 매핑

네트워크 인터페이스로 위장된 주식 정보:
- **Interface Name**: 종목명 (예: eth0, wlan0, br0)
- **Status**: 항상 "UP" (거래 중)
- **Speed**: 고정값 "1000 Mbps"
- **RX/TX Throughput**: **실제 주가** (Mbps 단위)
- **Packets**: 거래량 기반 생성값
- **Errors**: 0 (정상 상태)

## 설치 및 실행

### 주요 기능 설명

#### 🔍 주가 확인 (옵션 1-8)
- **[1] Interface Status**: 전체 종목 주가 요약표
  - 현재가, 변화량/퍼센트, 고점/저점, 상태, 거래량
  - 한 번에 모든 종목 확인 가능
- **[2] Specific Interface**: 개별 종목 상세 정보
  - 현재가, 변화량 및 퍼센트
  - 고점/저점 표시
  - 일봉 범위 내 현재 위치 (%)
- **[6] Traffic Monitoring**: **실시간 주가 모니터링** (가장 유용!)
  - 30초마다 자동 업데이트
  - 1번과 동일한 표 형식으로 실시간 표시
  - 모든 종목의 변화를 지속적으로 추적
  - Ctrl+C로 종료

#### 🔧 관심종목 관리 (옵션 9-13)
- **[9] Add Interface**: 새 종목 추가
  - 종목명 검색 또는 티커 직접 입력
- **[10] Remove Interface**: 기존 종목 제거  
- **[11] List All**: 현재 등록된 모든 종목 확인
- **[12] Reset Configuration**: 기본 9개 종목으로 초기화
- **[13] Backup/Restore**: 설정 백업 및 복원 관리

### 종목 추가 방법

#### 방법 1: 종목명으로 검색
```
[9] 선택 → [1] 선택 → "삼성" 입력 → 원하는 종목 선택
```

#### 방법 2: 티커 코드 직접 입력
```
[9] 선택 → [2] 선택 → "005930" 입력
```

## 설정 파일 관리

```
[13] 선택 → [1] 백업 생성

[13] 선택 → [2] 백업에서 복원
→ 기존 백업 파일 목록에서 선택
```

## 팁과 트릭

### 💡 효율적인 사용법
- **빠른 확인**: 옵션 1로 모든 종목을 한눈에 확인
- **지속 모니터링**: 옵션 6으로 실시간 변동 추적 (같은 표 형식)
- **상세 정보**: 옵션 2로 개별 종목의 자세한 분석
- **즐겨찾기**: 자주 보는 종목은 기본 9개에 포함시키세요
- **종목 관리**: 옵션 11로 현재 등록 종목 확인
- **정기 백업**: 좋은 포트폴리오 구성시 백업 생성

### 🔧 고급 기능
- **검색 활용**: 정확한 종목명 몰라도 일부만 입력해서 검색
- **인터페이스 이름**: 새 종목은 자동으로 eth5, wlan2 등으로 할당
- **오류 처리**: 잘못된 티커 입력시 자동 검증 및 오류 메시지
- **설정 유지**: 컴퓨터 재시작해도 관심종목 자동 복원

## 문제 해결

### 설정 파일 관련 문제
- **"Configuration file not found"**: `network_interfaces.json` 파일이 없습니다
  - 프로그램과 같은 폴더에 JSON 파일이 있는지 확인
  - 옵션 12로 기본 설정을 다시 생성
- **JSON 파일 오류**: 파일 형식이 잘못되었을 수 있습니다
  - 백업에서 복원하거나 초기화 실행

### 종목 데이터가 안 나올 때
- 장 마감 시간이거나 휴장일일 수 있습니다
- 인터넷 연결을 확인하세요
- 티커 코드가 정확한지 확인하세요

### 프로그램이 느릴 때  
- 첫 실행 시 데이터 로딩에 시간이 걸릴 수 있습니다
- 너무 많은 종목을 추가하면 느려질 수 있습니다
- 잠시 기다려주세요

### 검색이 안 될 때
- 정확한 종목명 일부를 입력하세요
- 한글로 입력하세요 (예: "삼성", "현대")
- 너무 짧은 검색어는 피하세요
